# teletext-decoder-font

I created this font (Teletext Decoder Regular) for use in the 'ttxweb' teletext-to-html decoder project; this font is based upon inspirations taken from (1) the pixel font depicted in the ETSI Teletext standard documents, (2) the font found in the CGROM of dozens of teletext decoders since the late 1980s in several variations, (3) the Teletext font found in the OpenSUSE 'tv-fonts' package (and also used by VLC Player), creating a mashup with some glyph additions and variations to the author's own taste.

In this repo, I will share the FontForge project file for the TTF file (Teletext Decoder Regular).

Note that for shipping with ttxweb, I further slimmed down the TTF file after exporting it with FontForge, mainly by limiting it to the following glyph ranges:
02C0-02DF, 2010-266F

Have fun teletexting!
